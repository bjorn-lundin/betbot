SYSTEM PREREQUISITES
====================

If time zone setting is needed
------------------------------
Due to problems with Python datetime module (its inability to handle 
time zones correctly), set the production system time zone to 
'Europe/Stockholm' to match ATG in Stockholm, Sweden.

1. Check that hw clock uses UTC -> /etc/default/rcS -> UTC=yes
2. What is the current tz? -> /etc/timezone
3. Set new tz -> sudo dpkg-reconfigure tzdata

CET -> Europe/Stockholm
UTC -> Etc/UTC

Subversion
----------
Install Subversion package
    sudo apt-get install subversion

Python
------
Make sure that a Python envionment is present
    python -c "import sys; print(sys.version)"
    ->
    2.7.3 (default, Aug  1 2012, 05:14:39)
    [GCC 4.6.3]

Install python-virtualenv if not in place
    sudo apt-get install python-virtualenv

Install psycopg2 system dependencies
    sudo apt-get install libpq-dev python-dev

Create virtual environment
  	virtualenv ppm_py_env

Install packages (latest)
  pip install --ignore-installed --requirement=pip.txt


Postgresql
----------
Install database
    sudo apt-get install postgresql

Create user (owner) and databases for test and production
    sudo su - postgres
    psql
    CREATE USER db_user WITH PASSWORD 'xyz';
    CREATE DATABASE db_test OWNER db_user;
    CREATE DATABASE db_prod OWNER db_user;

APPLICATION CONFIGURATION
=========================
Configuration file (conf.py)
    Copy conf_template.py to conf.py
    Enter all parameters into conf.py 

Virtual Environment path
------------------------
Make sure run.sh contains the correct Python VIRTUAL_ENV path,
e.g. -> export VIRTUAL_ENV="/home/joabi/xxx_py_env"
